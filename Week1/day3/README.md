# Titanic Dataset Analysis with Pandas

This project contains Pandas code snippets solving data analysis questions using the Titanic dataset.

## Overview

The script performs the following data queries:
1. **First Class Count:** Finds the number of passengers in First Class.
2. **Male Count:** Counts the total number of male passengers.
3. **Cabin C85 Count:** Determines how many passengers stayed in cabin C85.
4. **Female Survivors:** Generates a list of all female passengers who survived.
5. **Average Age of Non-Survivors:** Computes the mean age of passengers who did not survive.
6. **High Fare Passengers:** Retrieves the names of passengers who paid a fare greater than $200.
7. **Embarked from S:** Counts passengers who embarked from port 'S'.
8. **Senior Survivors:** Counts passengers older than 60 years who survived.
9. **Solo Travelers:** Filters details for passengers traveling with no siblings and no parents on board (`SibSp == 0` and `Parch == 0`).
10. **Oldest Passenger:** Identifies the details of the oldest passenger on board.

* Python 3.x
* Pandas
