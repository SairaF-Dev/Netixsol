# Statistics CLI Tool

## About the Project
A simple and efficient command-line tool written in Python to perform basic statistical analysis. This project helps in quickly calculating the **Mean, Median, Mode, Maximum, and Minimum** values from a list of numbers provided by the user.

## Key Functionalities
* **Statistical Math:** Calculates descriptive statistics (Mean, Median, Mode) without using heavy external libraries.
* **Error Handling:** Built-in validation to ensure the program handles non-integer inputs without crashing.
* **Efficient Logic:** Uses clean loop structures and dictionary mapping to process data quickly.

## How to Use
1. Make sure you have Python installed.
2. Run the script in your terminal:
   ```bash
   python day1_stats_cli.py

1. Enter your numbers separated by spaces (e.g., 10 20 30).
Example
Input: 10 20 20 30 40

Output:
```text
Mean: 24.0
Median: 20
Mode: 20
Maximum: 40
Minimum: 10
```
